==============Content================
RTC current date를 받는 인터럽트 종류를 찾아보았습니다.

여기서 날짜를 읽어올 때 인터럽트를 호출하면 자동으로 특정 레지스터에 저장이 되는건가요, 아니면 직접 레지스터를 초기화 하고 해당 레지스터에 넣어주는건가요??

받아올 때 팁 부탁드립니다..



==============Comment================
Get RTC DATE 인터럽트를 호출하면 해당하는 년,월,일 이 지정된 레지스터로 BCD 형태로 저장 됩니다.  해당 인터럽트의  서비스 설명에 가보시면 어느 레지스터에 저장되는지 아실수 있을 겁니다.

저장된 BCD 형태의 값을 아스키문자로 변환한 후 사용해야 하나요? 변환하고 사용하는 사례의 링크가 혹시 있을까요..?

한글자씩 출력한다면 BCD 형태에서도 값을 가져온뒤 추가적인 작업을 거치면 출력할수는 있을것같습니다. 그런데 저는 String 출력으로 진행하려고 BCD 형태에서 ASCII 형태로 변환후 String 출력 Interrupt를 사용하였습니다. 링크는 잘 모르겠습니다.  2자리수 BCD형태를 ASCII 형태로 바꾸기 위해서는 우선 첫번째 자리수 상위 4비트만 구한뒤 '0'(zero)의 ascii 코드를 더해주시고 하위 4비트를 구한뒤 '0'의 ascii 코드를 더해주시는 방법으로 년, 월, 일 모두 진행하시면 됩니다.
