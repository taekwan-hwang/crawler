==============Content================
과제 명세서에 나온 방법으로 bochs ~ nasm 까지 설치 완료 후
ssuos 소스에서 make, make run 실행시 '오류1' 과 같은 오류가 계속해서 발생하게 됩니다.
우분투 재설치 및 sudo 를 해당 명령어를 실행해봐도 같은 증상이 나타나네요.

오류 메세지를 살펴보니 bochs가 제대로 설치된 것 같지 않아 bochs 설치 폴더로 가보니 '오류3'  처럼 bochs 폴더가 생성되어 있지 않았습니다.

결국 
bochs 설치 문제인 것으로 추측 되어 다른 설치 방법이 있나 검색해보았으나 인터넷에 나와있는 설치방법 역시 과제명세서에 나온 방법과 크게 다르지 않았습니다.

질문의 내용을 정리하면
1. make run 시 발생하는 오류가 bochs로 인한 오류가 맞는 것인지
2. bochs 문제라면 설치가 온전히 안된 것으로 생각되는데 make && make install 외에 다른 설치방법 / 해결방안이 있는지 궁금합니다.

폴더가 정상적으로 생성되었습니다.
제대로 인스톨이 되었는지는 /usr/share/bochs 폴더가 생성되었는지로 알수있습니다.
