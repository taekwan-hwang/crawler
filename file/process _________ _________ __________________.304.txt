지금 Priority array는 잘 돌아가는 상태인거같은데.. 

Context switch를 어떻게 해줄지에서 막혀있습니다.

맨 처음 context switch가 발생하면
cur_process = proc[0](idle); 
next_process = 우선순위가 가장 높은 process 구조체포인터

switch_process(cur, next);를 통해서 context change를  해주고 있는데..

context change를 해준다는 말이.. 선택된 프로세스를 실행시키는 것이라고 생각하는데요..
제가 생각했을 때, proc_create에서 top 에서 2번째 떨어진 값에 해당 프로세스의 함수의 주소값이 저장되어있는거같은데..
이걸 실행시키는게 맞는건가요?

만약 이걸 실행시키는게 맞다면  어떻게 해야하는 지 . .. 참고할만한 개념이 있을까요?



==============Comment================
글쓴이 분께서 살짝 핀트를 잘못 잡고 계신것 같아서 몇 가지 도움드리겠습니다.
1. context switching과정은 생각하신 부분이 맞습니다. 과정은
기존 프로세스 -> 0번 프로세스
schedule() 함수에 의해서 다음에 수행할 프로세스 선택
0번 프로세스 0-> 다음에 수행할 프로세스
이런 과정으로 수행됩니다.
2. switch_process() 함수(매크로)를 사용하게 되면 함수의 주소값까지 생각할 필요 없이 문맥전환이 이루어지므로 이번 과제에서는 이 정도 깊이까지 고려하지는 않으셔도 됩니다.
