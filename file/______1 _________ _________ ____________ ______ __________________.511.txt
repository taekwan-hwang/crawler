==============Content================
몇시간동안 한 글자라도 출력하려는데 계속 검은 화면만 보이네요

CLS 함수 밑에 새 함수 만들고

al에 'A'넣고 cx에 1, ah에 09h, 0eh, 0ah, 13h 다 시험해 봤는데 계속 검은화면이거나 회색화면에 창 크기까지 바뀌네요

int 10h 사용하는 것같은데 너무 안되서 질문 올립니다.



==============Comment================
새 함수 위치가 잘못된거 아닐까요? 저도 함수 위치를 CLS 함수 위에 두니까 글자가 출력되더군요.

출력 방법은 자유겠지만 1번의 경우 인터럽트를 쓰지 않고도 출력 가능합니다 
보다 쉬운 방법이 있으니 비디오 메모리와 관련된 정보를 찾아보는게 좋을 것 같습니다

윗분처럼 CLS 함수 밑에 새 함수를 만들어 사용하는 경우 CLS 함수에서 사용하는 bx를 그대로 사용하신다면 offset이 잘못되어 엉뚱한 위치에 값을 저장하는 경우가 있습니다. 확인해보세요.

바이오스인터럽트 10번 사용하실때 제일먼저 비디오모드 설정을 하셔야합니다.
