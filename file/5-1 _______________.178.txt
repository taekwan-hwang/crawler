값 할당중에

'만일 page directory 탐색 중 page table 또한 존재하지 않으면 palloc_get_page() 함수를 사용하
여 page 하나를 할당 받고 page table로 사용'

라는 부분을 구현해서 돌렸는데요

되는줄 알고 했다가 뭔가 이상해서 테스트했더니
'=palloc_get_page()' 부분에서 계속 page_fault가 뜨는데 (이후 문장을 실행을 못시킵니다)

할당 자체를 잘못한거 같긴 한데 원인을 잘 모르겠습니다.
원인에 대한 힌트나 해결방안에 대한 힌트를 부탁드립니다.

일단 page_fault의 fault_addr은 C0002000에서 멈추지 않고 1000씩 계속 증가중이긴 합니다.

pd값은 cr3를 받아오는 함수를 이용하여 받아왔구요. 페이지 디렉토리 인덱스도 주어진 함수로 분리하여 사용하였습니다.
