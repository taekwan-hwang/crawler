4GB의 메모리를 인식하기 위해서 1024개의 페이지 디렉토리가 있으면
각각에 따른 페이지 테이블이 1024개 씩 있어
총 1024 * 1024 * 4Byte = 4MB의 테이블 공간이 필요합니다만, (4KB 테이블이 1024개)
코드 상 페이지 테이블이 1개 뿐입니다.
필요할때마다 페이지 테이블을 추가하는 형식으로 구현하면 되는건가요?

+추가질문입니다.
실제메모리에 테이블과 디렉토리를 추가했는데
페이지 디렉토리p비트를 1로하니 page_fault가 안일어나고 0으로하니 테이블 p비트에 상관없이 page fault가 반복됩니다
page_fault가 일어나는 조건이 페이지디렉토리 p비트가 0이고 (and) 페이지테이블의 p비트가 0일때가 맞나요?
제가 인덱스 계산을 잘못해서 그런걸까요, 아니면 조건이 저게 아닌가요
