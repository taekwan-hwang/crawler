==============Content================
저는 CLS에서 80*25*2만큼 0을찍어 화면을 지워주는 부분을 인용하여
반복문으로 문자열의 문자를 하나씩 출력하는 방법을 생각했는데요.
실행결과가 계속 까만화면으로 나오며 해결이 안되서
게시판에 있는 1번과제 질문들을 읽어보거나 구글링을 하며
최대한 잘못된 부분을 찾아보려고 해도 좀처럼 발견하기가 힘드네요.
아래는 제 코드 구조 입니다.

...
loop CLS

[msg 배열에 "Hello World" 저장]

[bx 0으로 초기화]
[cx에 msg 배열의 크기 대입]

PRINT:
[al, byte 이용하여 ax에 문자 하나씩 대입]
mov [es:bx], ax
add bx, 1
loop PRINT

loop 돌기전에 문자 하나만 대입해서 출력되는지 확인해보셨나요?

저도 계속 화면에 변화가 없더라구요.
부팅 과정에서 어셈블러에게 코드의 위치를 알려주는 org	0x7c00 명령어를 코드 상단부분에 입력한 후에 출력이 정상적으로 됐습니다.
혹시 저와 같은 문제일수도 있어 댓글남깁니다.

msg의 위치를 제대로 가져오기 위해서 시작지점을 boot loader 코드위치로 바꿔주는 org를 걸어주는게 필요하구요, 문자출력시에 한 문자는 1바이트로 표현되는 것이 아니라 문자 1바이트와 배경/글자색을 나타내는 1바이트의 2바이트로 표기됩니다.
