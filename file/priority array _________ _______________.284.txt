==============Content================
proc_create호출 시에

priority 이차원 배열에 Linked list 구조의 리스트 값을 저장하기 위해 먼저 초기화를 하고
넘겨받는 priority 값에 매칭되는 인덱스 위치에 list값을 저장하기 위해 row, col을 구한 후 해당 위치에 값을 저장하는 과정을 거친다고 이해했습니다.
추가적으로 이러한 priority 배열을 해당 과정에서 완성시키고,  active나 expired는 포인터 개념으로 사용하여 create과정이 아닌 나중에 배열 탐색과 swap과정에 사용하는 것으로 이해했습니다.

질문1: 우선 위의 흐름이 맞나요?
질문2: 처음 초기화 부분에서 list.h, list.c의 적절한 함수를 호출하는데(어떤 함수를 호출하는지는 알겠습니다.) 여기에 인자값으로 어떤 list를 넣어줘야는지 잘 모르겠습니다. 여기서 자꾸 오류가 생기네요. 리스트와 prirority array의 vector값이 일치하지 않는다는 점을 보아 서로 다른 구조를 가져 변환이 필요한거인가 싶기도한데, 단순 호출하는 리스트값의 문제인지 아니면 다른 이유인지 해결할만한 팁이 있다면 답변 부탁드립니다.
